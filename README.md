# University GatePass Tracker

A web-based system using **Java (JSP)** and **MySQL** to manage university gate passes.

## How to Run

1. Clone the repository  
   git clone https://github.com/bharathboienipalli/University-GatePass-Tracker.git

2. Open the project in NetBeans.

3. Import the database file:
database/gatepass_db.sql

4. Configure Apache Tomcat and MySQL.

5. Build and run the project.
